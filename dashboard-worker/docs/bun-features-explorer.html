Build & Compile

bun build --compile

Create standalone executables from your JavaScript/TypeScript code.

// Create a standalone executable
bun build --compile ./src/server.ts --outfile my-app

// Compile with optimizations
bun build --compile --minify ./src/app.ts --outfile app-optimized

// Target specific environment
bun build --compile --target=bun ./src/server.ts --outfile server

Bunx Package

bunx --package

Run packages with specific versions without installing them globally.

// Run TypeScript compiler with specific version
bunx --package typescript@5.0.0 tsc --version

// Use Prettier without installation
bunx --package prettier@3.0.0 --check src/**/*.ts

// Execute create-react-app without npx
bunx --package create-react-app@5.0.0 my-app

HTML Serving

bun ./**/*.html

Serve multiple HTML files easily with Bun's built-in server capabilities.

// Serve all HTML files in current directory
bun ./*.html

// Serve HTML files recursively
bun ./**/*.html

// Serve specific HTML files
bun index.html about.html contact.html

// Serve with custom port
bun --port 3000 ./*.html

Package Updates

bun update --interactive

Manage your dependencies with interactive package updates.

// Interactive package updates
bun update --interactive

// Check for available updates
bun update --check

// Update all dependencies
bun update

// Update specific package
bun update react

// Update to latest versions
bun update --latest

Publish & Versioning

bun publish & Version Management

Publish packages to npm and manage semantic versioning with Bun.

// Publish a package to npm
bun publish

// Publish with specific tag
bun publish --tag beta

// Dry run to test publishing
bun publish --dry-run

// Version management commands
bun version patch # v1.0.0 ‚Üí v1.0.1
bun version minor # v1.0.0 ‚Üí v1.1.0
bun version major # v1.0.0 ‚Üí v2.0.0

// Version with message
bun version minor -m "New features added"
