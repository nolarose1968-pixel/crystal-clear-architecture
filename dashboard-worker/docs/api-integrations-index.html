🔧 Bun Runtime Capabilities

📦
Version Information
Access Bun runtime version and build information 
// Get Bun runtime version
const bunVersion = Bun.version;
// => "1.0.29"

// Get git commit hash
const gitRevision = Bun.revision;
// => "f02561530fda1ee9396f51c8bc99b38716e38296"

// Get entry point path
const entryPoint = Bun.main;
// => "/path/to/your/script.ts"
⚡
Performance & Timing
High-precision timing and performance measurement 
// High-precision timing
const startTime = Bun.nanoseconds();
// ... your code ...
const duration = Bun.nanoseconds() - startTime;

// Async sleep
await Bun.sleep(1000); // 1 second
await Bun.sleep(new Date(Date.now() + 1000));
🔍
Inspection & Debugging
Advanced object inspection and debugging tools 
// Object inspection
const obj = { foo: "bar" };
const inspected = Bun.inspect(obj);

// Table formatting
const table = Bun.inspect.table([
  { name: "Alice", age: 30 },
  { name: "Bob", age: 25 }
]);
🗜️
Compression & Encoding
Built-in compression and encoding utilities 
// GZIP compression
const data = Buffer.from("Hello World".repeat(100));
const compressed = Bun.gzipSync(data);
const decompressed = Bun.gunzipSync(compressed);

// DEFLATE compression
const deflated = Bun.deflateSync(data);
const inflated = Bun.inflateSync(deflated);

New commands to access Bun runtime versioning information:

Command Purpose Output Status

bun run version:runtime Get complete runtime environment info Bun + Node.js + system information Active
bun run version:full Get complete version information Application + Runtime + Environment Active
bun run version:compare Compare versions across environments Version differences and compatibility Active
