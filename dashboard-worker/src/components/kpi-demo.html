üìö Usage Examples

Basic Usage:

import { createKPICard } from './kpi-component.ts'; const kpi = createKPICard({ label: 'Pending Amount', value: 125000, format: 'currency', color: 'warning' }); document.body.appendChild(kpi);

Pre-built KPIs:

import { createPendingAmountKPI, createTotalAgentsKPI } from './kpi-component.ts'; const pendingAmount = createPendingAmountKPI(125000); const totalAgents = createTotalAgentsKPI(42); document.body.appendChild(pendingAmount); document.body.appendChild(totalAgents);

Interactive KPI:

import { KPIComponent } from './kpi-component.ts'; const kpi = new KPIComponent({ label: 'Clickable KPI', value: 1000, clickable: true, onClick: () => alert('KPI clicked!') }); document.body.appendChild(kpi.render());
